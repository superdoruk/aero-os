# AERO OS 🚀

Açık kaynak, iki modlu Linux tabanlı işletim sistemi.

## Nedir?
AERO OS, kullanıcıya iki farklı mod sunar:
- **AEROWS** — Grafik arayüz, kolay kullanım
- **AEROUX** — Terminal öncelikli, güvenli mod

## Özellikler
- İzin yöneticisi (her uygulama için onay)
- Dahili antivirüs (ClamAV)
- Task bar konumu seçimi
- AERO temalı açılış ekranı

## Dosyalar
- `aero-welcome.sh` — Karşılama ekranı
- `aero-izin.py` — İzin yöneticisi
- `aero-tarama.sh` — Antivirüs tarama
- `aero-taskbar.sh` — Task bar ayarı
